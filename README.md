# 🤖 Agente Resolutor de Ecuaciones de 2do Grado

Este agente inteligente ayuda a estudiantes y profesionales a resolver ecuaciones cuadráticas de forma rápida, mostrando el paso a paso y utilizando la fórmula general.

## ✨ Características
- Resolución automática mediante la fórmula general.
- Formateo matemático profesional.
- Interfaz amigable para dispositivos móviles.

## 🛠️ Tecnología
- **Plataforma:** Voiceflow.
- **Lógica:** JavaScript/API interna.
- **Renderizado:** LaTeX / Unicode.

## 🚀 Cómo usarlo
1. Descarga el archivo `.vf` de este repositorio.
2. Impórtalo en tu workspace de [Voiceflow](https://www.voiceflow.com/).
3. ¡Dale a 'Run' y empieza a resolver!
